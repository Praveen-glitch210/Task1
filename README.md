# SMART LIGHT — LED Control Using Arduino and Mobile App

The **Smart Light System** is an IoT-based project that allows a user to control an LED light wirelessly using a mobile application and a microcontroller such as an Arduino board.
The system uses wireless communication (Bluetooth or Wi-Fi) to send commands from a smartphone to the microcontroller, which then controls the LED.

This project demonstrates:

* Embedded systems
* Wireless communication
* Mobile app interfacing
* Home automation concepts
* IoT fundamentals


Components:

| Component                              | Quantity | Purpose                |
| -------------------------------------- | -------- | ---------------------- |
| Arduino Uno/Nano                       | 1        | Main controller        |
| LED                                    | 1        | Light output           |
| 220Ω Resistor                          | 1        | Protects LED           |
| Bluetooth Module HC-05 / ESP8266 Wi-Fi | 1        | Wireless communication |
| Breadboard                             | 1        | Circuit connections    |
| Jumper Wires                           | Few      | Wiring                 |
| Smartphone                             | 1        | User control           |
| USB Cable                              | 1        | Programming Arduino    |
| Power Supply                           | 1        | System power           |


Commands:

| Button | Command Sent |
| ------ | ------------ |
| ON     | 1            |
| OFF    | 0            |


# Output

* LED turns ON when ON button is pressed in mobile app.
* LED turns OFF when OFF button is pressed.
* Wireless communication operates successfully between phone and Arduino.
